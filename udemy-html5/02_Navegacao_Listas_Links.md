### Navegação

Destina-se (apenas) ao bloco principal de links de navegação.

~~~ html
<nav>...</nav>
~~~

### Listas

Lista não ordenada:

~~~ html
<ul>
	<li>Item 1</li>
	<li>Item 2</li>
</ul>
~~~

Lista ordenada:

~~~ html
<ol>
	<li>Item 1</li>
	<li>Item 2</li>
</ol>
~~~

### Links

- Ligações internas: páginas do próprio site;
- Ligações externas: páginas de outros sites.

~~~ html
<a href="minha-pagina.html">Minha Página<\a>
~~~
